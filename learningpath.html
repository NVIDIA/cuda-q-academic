Launch Instructions

To access this project on qBraid:

  1. Log in to your qBraid account.
  2. Click Explore in the left-hand menu.
  3. Select the CUDA-Q Academic project card.

Launch Instructions

To access this project on Brev:

  1. Log in to your Brev account.
  2. Click the Deploy Launchable button.
  3. Click Go to Instance Page.
  4. Wait for the GPU to spin up, then select Open Notebook when it turns green.

Launch Instructions

To access this project on AWS Braket:

  1. Log in to your AWS account. New to AWS? Get started here.
  2. Launch a Braket Notebook instance.
  3. In the Launcher tab, click the CUDA-Q and Braket icon to navigate to the CUDA-Q examples folder.
  4. Follow the instructions in 1_CUDA-Q_Applications_Hub.ipynb / 2_CUDA-Q_Academic_library.ipynb.
  5. Navigate to the desired example.
