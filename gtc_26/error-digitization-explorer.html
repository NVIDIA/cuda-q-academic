Error Digitization Explorer

Investigate how measuring stabilizers digitizes a continuous Rx(θ) error into a correctable syndrome in the 3-qubit bit-flip code.

Instructions

  1. Enter a rotation angle θ and select which qubit receives the error, then click Apply Error
  2. Observe how fidelity degrades continuously with θ — the error is analog
  3. Step through the stabilizer checks to see how measurement digitizes the error into two branches
  4. Apply correction — both branches recover the original state, regardless of θ

Keyboard: Tab navigates; Enter activates buttons; arrow keys switch qubit selection.

Error Configuration
