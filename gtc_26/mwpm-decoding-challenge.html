MWPM Decoding Challenge

Explore how matching syndrome defects becomes harder as code distance and error complexity increase. Select a code distance and syndrome pattern, then try to find the minimum-weight perfect matching.

Instructions

  1. Select a code distance (d=7 or d=11) and a syndrome pattern (Easy or Hard).
  2. The lattice has toric (periodic) boundary conditions — the top edge connects to the bottom, and the left edge connects to the right. Dashed edges indicate wrap-around connections.
  3. Click any highlighted red defect node to select it, then click a second defect to pair them — a shortest path is automatically drawn.
  4. Continue pairing until all defects are matched.
  5. Click Check to evaluate your matching, or Show Solution to overlay the optimal MWPM (dashed blue).

Keyboard: Tab to navigate. Enter/Space to select/pair. Esc to deselect.
