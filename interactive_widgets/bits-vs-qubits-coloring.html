Bits vs Qubits: Vertex Coloring

Compare how classical bits and quantum qubits encode 2-colorings of a 7-vertex graph — one assignment at a time vs. all 128 simultaneously.

Instructions
  1. Classical side: Click the bit circles to toggle each vertex between color 0 (green) and color 1 (white). The graph updates to show your single chosen coloring.
  2. Quantum side: All 7 qubits start in superposition (Bloch sphere arrows on the equator). Press Measure to collapse the superposition — each qubit randomly becomes |0⟩ or |1⟩, producing one coloring on the graph.
  3. Press Measure repeatedly to sample different colorings. Notice how each measurement can produce a different result from the same superposition state.
  4. Press Reset Qubits to return all qubits to superposition.

Keyboard: Tab to navigate controls. Enter or Space to toggle bits or press buttons. Esc to reset qubits. ←→ arrow keys to navigate between bit toggles.

Classical Approach

A classical computer stores one coloring per bit string. To represent all 27 = 128 possible 2-colorings, you need 128 separate 7-bit strings — or 896 bits total.

Quantum Superposition

A quantum computer puts each qubit in superposition with a Hadamard gate. Just 7 qubits simultaneously encode all 128 colorings. Measurement collapses the superposition to a single coloring, each equally likely. Applying additional gates (as in QAOA) reshapes these probabilities so that colorings that solve the problem are sampled more often than others.
