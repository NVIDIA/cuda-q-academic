Hamming [7, 4, 3] Visualizer

A visualization of error correction using parity bits and matrix multiplication to detect and correct single-bit errors.

Instructions

  1. 1. Original Message — Toggle data bits D1–D4 (0 or 1) to set the 4-bit message. The codeword (7 bits: D1,D2,D3,D4,P1,P2,P3) updates automatically.
  2. 2. Codeword — Click any of the 7 bits in the codeword strip to inject an error (bit is highlighted in the error color). Click again to remove it. Add one error to see correction; add two or more to see undetected logical errors.
  3. Watch the Venn diagram and the Parity Check (H × r = s) to see how the syndrome identifies the flipped bit. Status shows HEALTHY or ERROR DETECTED.

Keyboard: Tab — move between sections. Left / Right — move between the 11 bit buttons (4 message bits, then 7 codeword bits). Enter or Space — toggle the focused bit. ESC — reset simulation.
