Interactive QFT Visualizer

Explore the Quantum Fourier Transform by interacting with phase rotations on the unit circle and observing the resulting matrix construction.

Instructions

  1. Click Visualize Column |0⟩ to run a guided animation that fills the first column of the QFT matrix using phase rotations on the unit circle.
  2. Click Visualize Column |1⟩ to fill the second column the same way. After that, the matrix switches to interactive mode.
  3. Click any empty cell to start filling a column. Once started, you must complete the column before moving to another. Use the unit circle to choose the correct phase: click a point, or use Arrow keys and Enter to confirm.
  4. Use Auto-Fill Remaining to complete the rest of the matrix, or Reset Matrix to start over.

Keyboard shortcuts: ESC — reset. Enter / Space — select or place. Arrow keys — navigate the unit circle when focused.
