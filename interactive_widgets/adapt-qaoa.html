ADAPT-QAOA vs. Standard QAOA

Interactive Max-Cut: build the ansatz layer-by-layer from the mixer pool (gradient-based selection).

Instructions

  1. ADAPT-QAOA: Each layer adds one cost block e−i HC γk and one mixer from the pool. Click the mixer with the largest gradient (green) to add it, then click "Minimize (γ, β)" to optimize. Repeat until the gradient norm is below threshold.
  2. Standard QAOA: Uses a fixed number of layers with the same mixer (Σ Xj) in every layer. Click "Run QAOA" to see an illustrative cost curve from initial ⟨HC⟩ (|+⟩^⊗n) to the optimal value (no real optimizer in this widget).
  3. Use "Reset All" to start over. The graph shows the Max-Cut instance (5 nodes); the goal is to minimize ⟨HC

The mixer pool Aj includes single-X, full QAOA mixer (Σ X), and two-qubit terms (XX, YY, YZ, ZY). ADAPT-QAOA selects the mixer with the largest gradient −i⟨ψ|[HC, Aj]|ψ⟩ at each step. Reference: CUDA-Q ADAPT-QAOA.

Keyboard: ESC to reset. Enter or Space to activate buttons.
