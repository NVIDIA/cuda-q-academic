Steane Code Noise Model Explorer

Simulate and correct quantum errors using Code Capacity, Circuit Level, and Phenomenological noise models.

Instructions

What to do: Place an error at one of the green dashed [E] boxes on the circuit (each marks a valid error slot). Then watch how the syndrome and correction change. Try different noise models to see where errors can occur.

  1. Choose a Noise Model: Code Capacity (data qubits only, before checks), Circuit Level (data or ancilla), or Phenomenological (extra slots between rounds).
  2. Mouse: Click any green dashed [E] box on the circuit to place or remove an error. Keyboard: Press Tab until the circuit is focused, then Enter to enter keyboard mode. Use Arrow keys to move between error slots (all valid locations in the current mode), and Space to place or remove an error.
  3. After placing an error, the Syndrome (S1, S2, S3) and Correction update. The decoder suggests which qubit to correct; if it matches the actual error, status shows Corrected.
  4. Press ESC once to clear the error or exit keyboard mode; press again to reset.

Keyboard: Tab — move to circuit (after dropdown). Enter — start moving between [E] slots. Arrow keys — next/previous error slot. Space — place or remove error. ESC — exit or reset.

Errors only occur on Data Qubits before checks run.
