Trajectory Noise Demo

Visualize a simple trajectory-based error model.

Instructions

  1. Set Error Probability P(error) with the slider (0–0.20). This is the probability of error on each qubit in the trajectory.
  2. For each of Qubit 1, 2, and 3, choose No Error (0) or Error (1) to fix the trajectory. The tree highlights the path that matches your choices.
  3. The tree shows all 2³ outcomes; each leaf shows the bit string (e.g. 111) and P(outcome), e.g. P(|111⟩) = probability of that outcome. Green = no-error path; red = path with at least one error.
  4. Press ESC to reset to default (P = 0.10, all qubits No Error).

Keyboard: Tab to move between slider and buttons. ESC — reset. Enter or Space — activate focused button. ←/→ to move between No Error / Error buttons. Arrow keys adjust the slider when it is focused.
