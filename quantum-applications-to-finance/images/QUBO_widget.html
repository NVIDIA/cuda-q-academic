QUBO Portfolio Optimization

Explore how risk (β) and return (α) weights affect the QUBO matrix and optimal portfolio selection.

Instructions

  1. Adjust the α (Return) and β (Risk) sliders to change the QUBO objective. The matrix and optimal portfolio update in real time.
  2. Compare the selected assets and portfolio risk/return. Lower QUBO quality (minimized) corresponds to the chosen trade-off.

Keyboard: Tab to move between sliders (focused slider has a green ring). Enter/Space to activate. ESC to reset if applicable.

QUBO matrix heatmap (Q)

Green cells = negative values (lower cost, favorable for the portfolio). Red cells = positive values (higher cost, unfavorable). Numeric values are displayed in each cell for accessibility.

QUBO matrix definition:

Q[i,i] = −αμ_i + βΣ_ii

Q[i,j] = 2βΣ_ij for i < j
